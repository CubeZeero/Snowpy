# Snowpy
Spotify Now Playing

Spotifyで再生中の楽曲をTwitterに自動で投稿できます

## ビルド
開発環境は HSP 3.5

**s_key.txt**と**t_key.txt**に

```
Client ID (API key)
Client Secret (API Secret)
```

を書き込み保存

**実行ファイル自動作成**からコンパイルしてください

暗号化されpackfileが生成されます
